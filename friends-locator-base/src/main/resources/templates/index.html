Friends Locator

Веб-сервис, который позволяет друзьям отслеживать местоположение друг друга.

Создать группу

Шаг 1

Создайте группу, участники которой будут отслеживать местоположение друг друга. В результате создания группы вы получите уникальный идентификатор, с помощью которого сможете приглашать в группу друзей.

Шаг 2

Пригласите в группу всех, с кем вы хотите обмениваться местоположением. Для этого сообщите им идентификатор группы или отправьте соответствующую ссылку для подключения.

Шаг 3

Следите за перемещениями участников группы на карте. Координаты участников обновляются каждые 30 секунд.
